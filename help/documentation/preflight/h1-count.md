# Verifica preliminare opportunità conteggio H1

![Verifica preliminare opportunità conteggio H1](./assets/h1-count/hero.png){align="center"}

L’opportunità di conteggio Verifica preliminare H1 in Sites Optimizer consente di identificare e risolvere i problemi relativi all’utilizzo delle intestazioni H1 sul sito web. Questa opportunità si concentra sulla valutazione della struttura del sito e fornisce consigli utili per un miglioramento.
