# Opportunità di verifica preliminare dei metadati

![Verifica preliminare opportunità metadati](./assets/metadata/hero.png){align="center"}

L’opportunità di Verifica preliminare dei metadati in Sites Optimizer consente di identificare e risolvere i problemi relativi all’utilizzo dei metadati sul sito web. Questa opportunità si concentra sulla valutazione della struttura dei metadati del sito e fornisce consigli utili per un miglioramento.
